# メール返信テンプレート

> このテンプレートを使って、AIに「このテンプレートで返信を書いて」と依頼できます。
> 自分の文体に合わせてカスタマイズしてください。

---

## 社内メール（報告・連絡）

```
件名: [件名]

[相手の名前]さん

お疲れ様です。[自分の名前]です。

[本文: 1-3段落で要点を記述]

[必要に応じて箇条書きで詳細]
- [ポイント1]
- [ポイント2]
- [ポイント3]

[アクション依頼がある場合]
お手数ですが、[期限]までにご確認いただけますでしょうか。

よろしくお願いいたします。

[自分の名前]
```

## 社外メール（初回連絡）

```
件名: [件名]

[会社名]
[部署名] [役職]
[相手の名前] 様

お世話になっております。
[自社名]の[自分の名前]と申します。

[本文: 目的と背景を簡潔に]

[提案内容や依頼事項]

ご多忙のところ恐れ入りますが、
ご検討いただけますと幸いです。

何卒よろしくお願い申し上げます。

---
[署名]
[自社名] [部署名]
[自分の名前]
TEL: [電話番号]
Email: [メールアドレス]
```

## フォローアップメール

```
件名: Re: [元の件名]

[相手の名前]さん/様

お忙しいところ失礼いたします。
[日付]にお送りした[件名/内容]の件について、確認させていただきたくご連絡いたしました。

[状況の確認 or 追加情報]

お手すきの際にご返信いただけますと幸いです。

[締めの挨拶]

[自分の名前]
```

---

## AIへの依頼方法

```
このテンプレートを使って、以下のメールの返信を書いてください:

相手: [名前]
内容: [メールの要旨]
トーン: [丁寧/カジュアル/フォーマル]
ポイント: [伝えたいこと]
```
